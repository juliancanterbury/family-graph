# Family Graph MVP - Shareable Version

This version is designed to be usable immediately by non-technical family members.

## Features

- One-page app
- Family graph
- Build from photo
- Manual face boxes
- Name faces
- Add relationships
- People list
- Local browser saving
- Export/import JSON for sharing data

## How to install

Copy these files into the root of your `family-graph` repository:

- index.html
- style.css
- app.js
- birthday-photo.jpg

Commit:

`Add shareable face naming and relationship builder MVP`

## How your brother can use it

1. Open the GitHub Pages site.
2. Go to Build from Photo.
3. Add face boxes and names.
4. Add mother/father/partner/child relationships.
5. Export JSON and send it back to you.
6. You import the JSON to merge his work.
