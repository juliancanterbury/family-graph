How to use this MVP

  1. Open Build from Photo.
  2. Use the sample photo or choose another group photo.
  3. Click Add face box, then click a face in the image.
  4. Name that person. Existing names are reused automatically.
  5. Add core relationships: mother, father, partner, child.
  6. Open Graph to see the family graph update.
  7. Click Export data to send the family data to someone else.

Important: this version is designed to get family members contributing quickly. The next stage is Supabase so everyone edits the same shared database.
