Clean replacement for the installer version.

- Magic link login goes straight to the app.
- Creates a profile row on first login.

## Install

Copy these files into the repo root:

- index.html
- app.js
- style.css
- sample-photo.jpg if present

Keep your existing `config.js`.

Commit and push.
